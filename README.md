ğŸ“– About The Project

EduSpark is a single-page web application designed to democratize access to high-quality study materials. By leveraging the advanced capabilities of Google's Gemini 2.5 Flash API, it acts as an intelligent tutor that can instantly digest complex topics or heavy PDF documents and output structured learning modules.

Unlike generic chatbots, EduSpark provides a structured learning path:

Summary: Get the big picture first.

Q&A: Deep dive into conceptual understanding.

Flashcards: Memorize key terms via active recall.

Quiz: Test your knowledge with instant feedback.

âš™ï¸ Getting Started

Follow these steps to set up EduSpark locally on your machine.

Prerequisites

A modern web browser (Chrome, Edge, Firefox).

A Google Gemini API Key (Get it for free here).

Installation

Clone the repository

git clone [https://github.com/Irtaza404/EduSpark.git](https://github.com/Irtaza404/EduSpark.git)


Navigate to the project folder

cd EduSpark


Configure API Key
Open index.html in any code editor. Find the apiKey constant near the bottom script tag and insert your key:

const apiKey = "YOUR_GEMINI_API_KEY_HERE";


Run the App
Simply double-click index.html to open it in your browser. No npm install or local server is required!
